# Hotel
//In Progress//
A Java Web Project as solution to a college activity. It's hotel management software that runs in a server with servlet, MVC architecture and JSP output.
The main idea of the project is to change the simulated database to a MySQL Database, inprove the JSP files and complete the main actions, they are:
* Login the employees and clients;
* Able a employee to view the available and not available rooms;
* Able a employee to view the reserved rooms of the clients;
* Able a employee to reserve a room to a client;
* Able a employee checkin and checkout a client;
* Able a employee to view the indoor client's bill;
* Able a client to view its reserved rooms;
* Able everyone to view the rooms information.
